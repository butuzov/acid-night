## Syntax Highlighting

Supported syntax highlighting: Go (and Go templates), Python, Starlark, PHP, TypeScript, JavaScript, CSS/LESS/SCSS, Dockerfiles, Markdown, protobuf, hcl, shell.


## Dev Install

```bash
# Compile, build and install
make install_ext_dev
```
